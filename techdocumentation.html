What is Turing?

Turing is a language developed in the University of Toronto by Ric Holt and James Cordy. It was created in 1982. It is very simple to use and understand compared to other programming languages taught to beginners.

Here are some advantages of Turing:

  • Low processor drain
  • Easily readable syntax
  • Free to use

    • Programming in Turing

    Turing is notable for its syntax. For example, here's the "Hello World" program in Java: 

    
        class HelloWorldApp {
          public static void main(String[] args) {
            System.out.println("Hello World!"); // Prints the string to the console.
          }
        }

    And here is the same code in Turing: 

     put "Hello World!"
    Specific Syntax

    Turing has some specific syntax that makes it different from other languages.

  • Variables are declared by typing the variable's name, a colon, then the variable's data type:
    • var name : string
  • User input is handled using the "get" statement:
    • get name
  • The "put" statement replaces "print" or "console.log" from other languages:
    • put name

    There are other differences found in this language. Visit compsci.ca for more information.

    Use in teaching

    Due to its simplicity, Turing has been taught in schools worldwide. One such school is William Lyon Mackenzie C.I, located in Toronto.

    Mackenzie, due to its MaCS program, has a wide variety of dedicated students. As such, they do some amazing things with Turing that you might not expect, even though it is thought of as a beginner-level language. Examples include trivia games, RPGs, and complex particle systems.
